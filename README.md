# 🎨 Comix Craft Zone

**Comix Craft Zone** is a powerful, intuitive digital comic creator designed for storytellers, artists, and hobbyists. Build your narrative with ease using drag-and-drop frames, customizable speech bubbles, and smart layout tools.

![App Preview](https://picsum.photos/seed/comic-app/1200/600?blur=2)

---

## 🚀 Key Features

### 🖼️ Dynamic Layouts
- **Algorithmic Generation**: Instantly create unique frame arrangements with a single click.
- **Pro Templates**: Choose from a library of classic comic and modern webtoon layouts.
- **Manual Control**: Move, resize, and rotate frames to fit your vision perfectly.

### 💬 Smart Bubbles & Text
- **Customizable Shapes**: Rectangle, Circle, Cloud, Thought, Shout, and more.
- **Adjustable Tails**: Drag and reposition tails to point exactly where you need.
- **Typography**: Full control over fonts, sizes, colors, and text shadows.

### 📏 Webtoon Stitching
- **Vertical Storytelling**: Seamlessly combine multiple images into a long-form webtoon layout.
- **Auto-Numbering**: Automatically number your frames for tutorials or sequential storytelling.
- **Smart Resizing**: Page height automatically adjusts to fit your content perfectly.

### 🛠️ Professional Tools
- **Multi-Page Support**: Manage complex stories across multiple canvas pages.
- **Asset Library**: Built-in stickers, frames, and bubbles, plus multi-image upload support.
- **Undo/Redo**: Experiment freely with a robust history system.
- **Export**: Save your creations in high-quality formats ready for sharing.

## ✨ Why Comix Craft Zone?

Whether you're a professional artist or just starting out, Comix Craft Zone provides the tools you need to bring your stories to life. Our goal is to remove the technical barriers to digital comic creation, allowing you to focus on what matters most: **your story**.

## 🇷🇺 Описание на русском

**Comix Craft Zone** — это мощный и интуитивно понятный конструктор цифровых комиксов, созданный для сторителлеров, художников и любителей. Создавайте свои истории с легкостью, используя перетаскиваемые кадры, настраиваемые текстовые облака и инструменты умной верстки.

### Основные возможности:
- **Динамические макеты**: Алгоритмическая генерация и библиотека профессиональных шаблонов.
- **Умные облака**: Полная настройка формы, хвостов и типографики.
- **Склейка (Stitch)**: Создание длинных вертикальных вебтунов с автоматическим выравниванием и нумерацией.
- **Библиотека ассетов**: Встроенные стикеры, рамки и поддержка массовой загрузки изображений.
- **Профессиональный контроль**: Панель свойств для точной настройки цветов, шрифтов и кадрирования.
- **Удобный рабочий процесс**: Многостраничность, система отмены/повтора (Undo/Redo), зум и качественный экспорт.

## 📦 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/comix-craft-zone.git
   ```
2. Navigate to the project directory:
   ```bash
   cd comix-craft-zone
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

---

## 📖 Usage Guide

1. **Choose a Layout**: Start with a template or let the algorithm generate a random frame set.
2. **Add Content**: Drag images into frames or upload your own from the "Uploads" tab.
3. **Tell the Story**: Add speech bubbles and text. Customize their style in the right-hand Property Panel.
4. **Stitch It Up**: Use the "Stitch" feature to create long vertical webtoon sequences.
5. **Export**: Once your masterpiece is ready, hit the "Export" button to save it.
